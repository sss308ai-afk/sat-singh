# Multi-AI Voice Typer (Windows 11 Setup)

यह एप्लिकेशन आपकी आवाज़ (हिंदी/अंग्रेजी) को पहचानकर Notepad, Word, और Excel जैसे ऐप्स में टाइप करने में मदद करता है।

## चलाने के निर्देश (Instructions to Run)

1. **Node.js इंस्टॉल करें:** अगर आपके कंप्यूटर पर Node.js नहीं है, तो [nodejs.org](https://nodejs.org/) से डाउनलोड करके इंस्टॉल करें।
2. **ZIP फ़ाइल निकालें (Extract):** डाउनलोड की गई ZIP फ़ाइल को किसी फोल्डर में निकालें।
3. **ऐप शुरू करें:** फोल्डर के अंदर मौजूद `run_app.bat` फ़ाइल पर डबल-क्लिक करें।
4. **ब्राउज़र में खोलें:** ऐप अपने आप आपके ब्राउज़र में `http://localhost:3000` पर खुल जाएगा।

## मुख्य विशेषताएं (Key Features)
- **Voice Typing:** हिंदी और अंग्रेजी दोनों में टाइप करें।
- **AI Integration:** 'सही करो' या 'अनुवाद करो' जैसे कमांड्स का उपयोग करें।
- **App Simulation:** Word और Excel जैसा अनुभव।
- **Export:** अपनी फाइलों को .txt, .md, या .csv में डाउनलोड करें।
